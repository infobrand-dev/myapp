# Meetra Documentation

**Meetra** adalah platform bisnis lengkap untuk UKM Indonesia — menggabungkan komunikasi pelanggan, akuntansi, manajemen tim, dan operasional bisnis dalam satu aplikasi.

Dokumentasi ini mencakup modul **Accounting** — pembukuan, laporan keuangan, penjualan, pembelian, stok, dan kas.
